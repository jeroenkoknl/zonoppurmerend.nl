## Aanpassing

Om de site naar behoren te laten werken, is [functions.php](./wp-content/themes/eco-press/functions.php) aangepast. Nadat het Eco Press theme is geupdate, moet deze aanpassing opnieuw worden toegepast.

De specifieke aanpassing is op regel 57:

```
			'Lato:300,300italic,700,700italic|Lato', 
```
